# Alemar11 Tap

## How do I install these formulae?

`brew install alemar11/tap/<formula>`

Or `brew tap alemar11/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "alemar11/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
